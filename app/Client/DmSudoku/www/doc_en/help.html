How to manage the menu

| New

When the light at the up-right corner is on (yellow), you can click the icon and create a new sudoku. The previous sudoku will be dicarded.

If the light is grey you have to wait until its change.

Copy

This button allow to copy an external sudoku (from a news paper, for example)

Load

It allows to load a sudoku previously saved with 'Save'.

Save

It allows to save the current sudoku. You can save until nine diferents sudokus.

When you save a new version of a sudoku on course, this new version overwrites the old one. This is, you only can have one version of each sudoku.

Sudokus saved can be restored with 'Load'.

| Change language

Clicking over the text, you can switch the application language.

Change level

You can up or down the difficulty level of a new sudoku clicking over plus or minus icons.

Pen - pencil

This buttom allows to change writting mode. You can write with pen or with pencil.

If you write with pen, numbers are shown in black. This numbers can not be erased with the grey brush (see below)

In case you write with pencil, numbers will be shown in grey. This allows use the grey brush to erase them.

| Clear

The icon is in grey when the write-system is 'with pencil'.

When the icon is grey and you click over it, only grey numbers will be erased. Otherwise all the numbers will be, that is, you will start the sudoku from the beginning.

Show mistakes

You can verify if you have written wrong numbers clicking over this icon. Wrong numbers will be marked in red. If you click one more the sudoku will appear in its usual state.

Solve

You can deceive yourself clicking here... or simply can verify that the sudoky have an effective solution. Any way, this icon shows the sudoku solved.

 

How to write numbers

You can write number from keyboardor, by mouse or by touchscreen.

From key board you can use:

  • To write numbers: The main typing keypad or the numeric keypad.
  • To erase numbers: Keys 'space','delete' or 'bakspace'.
  • To move: The directional keys.

By mouse or touchscreen you have to use the numeric panel at the bottom of the window.

  • Write numbers: Clicking on the numeric panel.
  • Erase numbers: Clicking on the blank key of the numeric panel.
  • To move: Clickin directly on the sudoku board.
